# sanyastem.github.io

Архив старого блога. Сайт переехал на **[sanyastem.dev](https://sanyastem.dev/)**.

Этот репозиторий теперь содержит только страницу-редирект: любой заход на
`sanyastem.github.io` автоматически перенаправляется на новый домен.

- `index.html` - анимированная страница с автопереходом (~4 c) и кнопкой «Перейти сейчас».
- `404.html` - мгновенный редирект для старых вложенных ссылок.
- `.github/workflows/pages.yml` - публикует статику в GitHub Pages без сборки Jekyll.

Новый блог - full-stack приложение (Angular SSR + ASP.NET Core), исходники
в отдельном репозитории.
