Как запустить сайт на GitHub Pages за 5 минут

GitHub Pages — бесплатный хостинг прямо из репозитория. Создаёшь репо с именем username.github.io, кладёшь туда index.html — и сайт уже в интернете. В этом гайде разберём весь процесс от нуля: создание репозитория, настройку ветки, кастомный домен и HTTPS.

01
Создай репозиторий

Назови его username.github.io — именно так, где username — твой логин на GitHub.

02
Добавь index.html

Это главная страница. Можно начать с минимального HTML — GitHub Pages сразу подхватит.

03
Включи Pages в настройках

Settings → Pages → Source: Deploy from branch → выбери main или master.

04
Подожди 1-2 минуты

GitHub задеплоит сам. Сайт появится по адресу username.github.io.

💡

Если изменения не появляются — проверь, что пушишь именно в ту ветку, которую выбрал в настройках Pages.
