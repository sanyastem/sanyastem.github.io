Адаптивный дизайн: media queries без боли

Большинство проблем с адаптивностью решаются не горами медиазапросов, а правильным подходом с самого начала. Разбираем mobile-first и современные CSS-инструменты.

Mobile-first — пиши для мобильных сначала

Базовые стили — для мобильных. Медиазапросы расширяют их для больших экранов. Это проще, чем переопределять десктопные стили под мобильные.

/* Базово — мобильный */
.grid {
  display: grid;
  grid-template-columns: 1fr;
  gap: 16px;
}

/* Планшет и выше */
@media (min-width: 768px) {
  .grid {
    grid-template-columns: 1fr 1fr;
  }
}

/* Десктоп и выше */
@media (min-width: 1024px) {
  .grid {
    grid-template-columns: repeat(3, 1fr);
  }
}

Стандартные точки перехода

/* Мобильный:  < 480px  — базовые стили */
/* Планшет:    ≥ 768px  */
/* Десктоп:    ≥ 1024px */
/* Широкий:    ≥ 1280px */

@media (max-width: 480px) { ... }
@media (min-width: 768px) { ... }
@media (min-width: 1024px) { ... }
💡

Используй min-width (mobile-first) вместо max-width — меньше переопределений и логичнее читается.

clamp() — адаптивные размеры без медиазапросов

clamp(min, preferred, max) задаёт значение которое плавно масштабируется между минимумом и максимумом. Идеально для шрифтов и отступов.

/* Шрифт: минимум 1.2rem, максимум 2.4rem, плавно между ними */
h1 {
  font-size: clamp(1.2rem, 5vw, 2.4rem);
}

/* Отступ: минимум 16px, максимум 80px */
.section {
  padding: clamp(16px, 5vw, 80px);
}

/* Ширина контейнера */
.container {
  width: min(1200px, 100% - 48px);
  margin: 0 auto;
}

Адаптивная сетка без медиазапросов

Комбинация auto-fill и minmax — сетка сама решает сколько колонок поместится:

.cards {
  display: grid;
  /* Каждая колонка минимум 260px, максимум равная доля */
  grid-template-columns: repeat(auto-fill, minmax(260px, 1fr));
  gap: 16px;
}
/* На мобильном — 1 колонка, на планшете — 2, на десктопе — 3+ */
/* И всё это БЕЗ медиазапросов */

Не забудь viewport meta

Без этого тега мобильные браузеры масштабируют страницу как десктоп:

<meta name="viewport" content="width=device-width, initial-scale=1.0">
⚠️

Никогда не используй фиксированные пиксельные ширины для контейнеров типа width: 960px — это ломает вёрстку на мобильных. Используй max-width + width: 100%.
