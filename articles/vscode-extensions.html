VS Code: расширения которые я использую каждый день

Из сотен расширений я выделил те, без которых реально неудобно работать. Никакого мусора — только то, что реально используется каждый день.

Prettier — форматирование кода

Prettier автоматически форматирует код при сохранении файла. Никаких споров о стиле — один стандарт для всей команды.

После установки добавь в settings.json:

{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode"
}
💡

Создай файл .prettierrc в корне проекта для кастомных настроек: ширина строки, одиночные/двойные кавычки, точка с запятой.

GitLens — суперсилы для Git

GitLens показывает прямо в коде кто, когда и зачем написал эту строку (blame-аннотации). Удобно смотреть историю файла, сравнивать ветки, изучать изменения.

Самое полезное: наводишь на строку — видишь последний коммит, автора и сообщение. Бесценно при разборе чужого кода.

Error Lens — ошибки прямо в строке

Error Lens выводит ошибки и предупреждения прямо рядом со строкой кода — не нужно смотреть в нижнюю панель. Экономит кучу времени.

Path Intellisense — автодополнение путей

Path Intellisense автоматически дополняет пути к файлам при написании импортов. Незаменим в проектах с глубокой структурой папок.

// Начинаешь писать путь — расширение показывает варианты
import styles from '../../components/ui/← тут автодополнение

Thunder Client — REST-клиент прямо в VS Code

Thunder Client — лёгкая альтернатива Postman прямо внутри редактора. Отправляй HTTP-запросы к API без переключения окон. Поддерживает коллекции, переменные среды, история запросов.

Auto Rename Tag — переименование тегов

Переименовываешь открывающий HTML-тег — закрывающий меняется автоматически. Мелочь, но бесит когда этого нет.

Настройки которые стоит включить сразу

{
  // Показывать пробелы и табы
  "editor.renderWhitespace": "boundary",

  // Подсвечивать текущую строку
  "editor.renderLineHighlight": "all",

  // Шрифт с лигатурами
  "editor.fontFamily": "'Fira Code', monospace",
  "editor.fontLigatures": true,

  // Автосохранение
  "files.autoSave": "onFocusChange",

  // Линейка на 80 символов
  "editor.rulers": [80]
}
💡

Открыть settings.json быстро: Ctrl+Shift+P → «Open User Settings (JSON)».
